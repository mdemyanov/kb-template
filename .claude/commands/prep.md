# Подготовка к встрече

Используй skill `meeting-prep` для подготовки к встрече.

## Параметры

- `vault_name`: {{VAULT_NAME}}
- `vault_tool`: aigrep
- `people_dir`: 10_PEOPLE
- `projects_dir`: 30_PROJECTS/active
- `committees_dir`: 20_MEETINGS/committees

## Алгоритм

1. Определи тип встречи:
   - 1-1 с сотрудником
   - Комитет
   - Проектная встреча
   - Внешняя встреча

2. Собери контекст:
   - Профиль человека / участников
   - История предыдущих встреч
   - Открытые договорённости
   - Связанные проекты

3. Подготовь повестку:
   - Темы для обсуждения
   - Вопросы, которые стоит задать
   - Предупреждения (просрочки, конфликты)

4. Сформируй бриф:
   - Ключевая информация
   - Рекомендации по подходу
   - Риски и точки внимания

## Использование

```
/prep [имя человека или название встречи]
```

Примеры:
- `/prep Иванов` — подготовка к 1-1 с Ивановым
- `/prep архитектурный комитет` — подготовка к комитету
- `/prep проект миграции` — подготовка к проектной встрече
