# Постобработка встречи

Используй skill `meeting-debrief` для обработки результатов встречи.

## Параметры

- `vault_name`: {{VAULT_NAME}}
- `vault_tool`: aigrep
- `people_dir`: 10_PEOPLE
- `templates_dir`: 90_TEMPLATES

## Алгоритм

1. Получи входные данные:
   - Транскрипт встречи (текст или файл)
   - Тип встречи (1-1 / комитет / проектная)
   - Участники

2. Извлеки ключевую информацию:
   - Основные темы обсуждения
   - Принятые решения
   - Договорённости (кто, что, когда)
   - Открытые вопросы

3. Сформируй документ:
   - Используй соответствующий шаблон
   - Заполни все секции
   - Добавь метаданные (frontmatter)

4. Сохрани в нужную папку:
   - 1-1: `10_PEOPLE/{id}/1-1/YYYY-MM-DD.md`
   - Комитет: `20_MEETINGS/committees/{id}/YYYY-MM-DD.md`
   - Сессия: `20_MEETINGS/sessions/YYYY-MM-DD_{title}.md`

5. Обнови связанные документы:
   - Профили людей (если есть важные наблюдения)
   - Проекты (если обсуждались статусы)
   - Приоритеты (если изменились)

## Использование

```
/debrief
```

После вызова команды:
1. Вставь транскрипт или заметки встречи
2. Укажи тип встречи и участников
3. AI обработает и создаст структурированный документ
