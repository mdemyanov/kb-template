# Постобработка встречи

Используй skill `meeting-debrief`.

Параметры vault передаются из секции "Контекст для skills" в CLAUDE.md.

## Использование

1. Вызови `/debrief`
2. Предоставь транскрипт или заметки встречи
3. Укажи тип встречи и участников (если не указано в источнике)

Skill извлечёт договорённости, решения и задачи, предложит обновления в vault.
